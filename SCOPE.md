## In Scope

The following capabilities and responsibilities are owned by this project:

- Generic web scraping and browser automation tools
- Search capability tools
- MCP (Model Context Protocol) tool server
- Tool registration and management

## Explicitly Out of Scope

Document-specific processing such as PDF parsing and smart extraction (handled by Anchor). Identity and authentication (handled by Zitadel/Aegis). Infrastructure provisioning (handled by Plinth).
