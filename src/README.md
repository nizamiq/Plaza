# Source Directory

This directory will contain the source code for Plaza MCP services.

## Planned Structure

```
src/
├── universal-scraper/     # Universal web scraping service
├── browser-automation/    # Playwright-based browser automation
├── web-search/           # Unified web search interface
├── data-enrichment/      # LinkedIn and data enrichment
├── intelligence-search/  # Intent detection and cross-referencing
└── shared/               # Shared utilities and types
```
